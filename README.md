# Requirement

-   Node: v16.17.0
-   NPM: 8.15.0
-   TS: 5.1.6

# Instalation

Go to the root directory and run the following commands:

1. npm i
2. npm run dev (to start development server)

# Setup

Go to file ".env" located in the root directory. Modify the API root URL if you need.
